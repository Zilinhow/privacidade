Política de Privacidade

Extensão Chrome: Classificador Eproc TJSP - Nobre.AI
Última atualização: 22 de maio de 2026

Resumo: Esta extensão é uma ferramenta de produtividade interna do escritório Nobre Advogados. Ela não coleta, armazena ou transmite dados pessoais para servidores externos. Todos os dados processados permanecem no navegador local do usuário.

1. Quem somos

O escritório Nobre Advogados é o desenvolvedor e único distribuidor desta extensão. A ferramenta foi criada para uso exclusivo dos advogados do escritório que possuem credenciais ativas nos sistemas integrados.

2. Quais dados a extensão acessa

A extensão acessa apenas localmente, no navegador do usuário, os seguintes dados:

  • Conteúdo da área de transferência: apenas quando o usuário cola texto estruturado em campos das páginas dos sistemas integrados. A leitura é em resposta a uma ação explícita do usuário (Ctrl+V).
  • Conteúdo das páginas dos sistemas integrados: apenas leitura, para identificar campos a preencher automaticamente. Não há captura ou armazenamento desses dados.
  • Configurações da própria extensão: armazenadas no localStorage do Chrome (preferências de uso, flags temporárias de fluxo).

3. O que a extensão NÃO faz

  • Não envia dados para servidores próprios da extensão
  • Não realiza tracking, analytics ou coleta de comportamento
  • Não armazena dados pessoais, financeiros ou de processos judiciais
  • Não compartilha dados com terceiros
  • Não modifica configurações do navegador além das próprias preferências

4. Sistemas integrados

A extensão automatiza tarefas em sistemas onde o usuário já possui credenciais legítimas:

  • Eproc TJSP (eproc.tjsp.jus.br, eproc1g.tjsp.jus.br) — Tribunal de Justiça de São Paulo, acessado via login OAB do advogado.
  • CRMJUR Ecodev (crmjur.ecodev.com.br) — Sistema de gestão jurídica contratado pelo escritório.
  • Nobre.AI Lovable (nobre-ia.lovable.app) — Aplicação interna do escritório.

O usuário é responsável por respeitar os Termos de Uso de cada um desses sistemas.

5. Permissões do Chrome

As permissões solicitadas no manifest são justificadas pelo funcionamento da extensão:

  • clipboardRead: leitura da área de transferência para colagem de dados estruturados.
  • scripting: injeção de scripts nas páginas dos sistemas integrados para automação.
  • activeTab: interação com a aba ativa quando o usuário clica no ícone da extensão.
  • host_permissions: apenas para os domínios listados na seção 4.

6. Alterações nesta política

Esta política pode ser atualizada a qualquer momento. A versão mais recente estará sempre disponível nesta URL. Recomendamos consulta periódica.

Para dúvidas sobre esta política ou sobre o funcionamento da extensão, entre em contato:
